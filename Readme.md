Про проєкт
Серверний Backend-додаток «To-Do List» API, розроблений на базі Django та Django REST Framework (DRF)

Реалізовані функції:
1. Кастомна модель користувача з автентифікацією через `Email` та `Password`.
2. Повний набір CRUD-операцій для керування списком завдань (ізольовано для кожного користувача).
3. Автоматична інтерактивна документація **Redoc** за стандартом OpenAPI.
4. Публічний ендпоінт "Про додаток" з описом та емблемою.
